# ⚡ Quick Start - Deploy STANDOUT for Free

**Goal**: Get your game online at a custom domain like `standout.hemant` in 20 minutes!

---

## 🎯 3 Simple Steps

### 1️⃣ Get Free Domain (5 min)
**Choose ONE:**

- **DuckDNS** (Easiest): https://www.duckdns.org
  - Sign in with Google/GitHub
  - Create: `standout-hemant`
  - ✅ Get: `standout-hemant.duckdns.org`

- **No-IP**: https://www.noip.com
  - Sign up (free)
  - Create: `standout-hemant.ddns.net`

- **Freenom**: https://www.freenom.com
  - Get free `.ga` or `.ml` domain
  - ✅ Get: `standout.ga`

---

### 2️⃣ Deploy to Hosting (10 min)
**Choose ONE:**

- **Render.com** (Recommended)
  1. Push code to GitHub
  2. Go to https://render.com
  3. New → Web Service → Connect GitHub
  4. Auto-detects from `render.yaml`
  5. ✅ Get: `your-app.onrender.com`

- **Railway.app**
  1. Go to https://railway.app
  2. New Project → Deploy from GitHub
  3. ✅ Get: `your-app.railway.app`

---

### 3️⃣ Connect Domain (5 min)

**In your domain provider (DuckDNS/No-IP/Freenom):**
1. Add CNAME record
2. Point to: `your-app.onrender.com` (or railway URL)
3. Wait 10-30 minutes
4. ✅ Done!

**Test**: Visit `https://standout-hemant.duckdns.org`
